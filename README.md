## Running the application


1. **Clone the repository** 

2. **Navigate to the root directory of the project** 

3. **Run backend** using dotnet in cmd
   - dotnet run
     
  default port 5078
     
5. **Run frontend** using npm
   - npm install
   - npm start

default port 3000
